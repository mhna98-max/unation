    🎰 룰렛 후원 설정

    방송 플랫폼 & 채팅 위젯

    방송하는 플랫폼을 선택하면 채널 페이지에 배지가 표시되고, 라이브 중 사용할 수 있는 채팅 위젯 주소가 발급됩니다.

    SOOP·치지직은 공식 공개 채팅 임베드를 제공하지 않아요. 본인 채널의 채팅 팝업 URL이나, Chazzy 같은 통합 채팅 뷰어 도구의 URL을 붙여넣어 사용할 수 있어요.

    방송 프로그램 세팅

    OBS소스 추가 → 브라우저 → 통합 알림창 URL 붙여넣기 → 크기 800x600 권장
    XSplit소스 추가 → 웹 페이지 → 통합 알림창 URL 붙여넣기 → 투명 배경 유지
    트위치채널 정보 패널에 후원 링크 URL을 등록하면 시청자가 후원 페이지로 이동합니다.
